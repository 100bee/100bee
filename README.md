### ITPT — IT Interview Practice Tool
> CS 기술 면접 준비 플랫폼

개발자를 위한 AI 기반 면접 연습 서비스입니다. 음성으로 답변하면 STT 변환 후 GPT-4o-mini가 채점·피드백·꼬리질문을 자동 생성합니다.

**주요 기능**
- Naver Clova Speech STT + OpenAI GPT-4o-mini 채점/피드백
- 카테고리별 통계 대시보드 (OS · DB · Java · Spring · Network)
- JWT 인증 + Google · Kakao · Naver OAuth 2.0 소셜 로그인
- 관리자 페이지 (문제 은행 CRUD, 유저 관리, 면접 기록)
- Docker Compose 기반 개발 환경 / AWS EC2 배포 예정

**Stack** `Spring Boot 3` `Java 17` `Spring Security` `JPA` `MySQL 8` `JWT` `OAuth 2.0` `GPT-4o-mini` `Clova STT` `React` `TypeScript` `Docker`

### Chronote — AI 기반 학습 관리 플랫폼
> AI 매칭 기반 그룹 스터디 플랫폼

AI가 관심사를 분석해 스터디 그룹을 연결하고, 타이머·투두·랭킹으로 꾸준한 학습 습관을 형성합니다.

**주요 기능**
- FastAPI 기반 AI 관심사 매칭 → 그룹 스터디 자동 연결
- 실시간 그룹 채팅 / 개인 학습 기록
- ⏱타이머 / 투두 시스템 + 랭킹 보드
- 학습 통계 데이터 시각화
- Node.js → Java/Spring Boot 마이그레이션 진행 예정

**Stack** `Node.js` `Express` `FastAPI` `Python` `MySQL` `MongoDB` `React` `SCSS`
